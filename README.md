### Just 3 steps to get started:
  1) **Install [C++ SFML Library](https://www.sfml-dev.org/download.php).**
  2) **In the "build.bat" file, specify your path to SFML.**
  3) **Run "run.bat".**

### Author's libraries used by this project:
